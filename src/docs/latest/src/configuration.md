# Configuration

## Prerequisites

* Rust Compiler
  ```sh
  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  ```

Or directly from their website: [Rust](https://www.rust-lang.org/tools/install)

## Installation

1. Install it from Cargo
    ```sh
    cargo install atlas_77
    ```
2. Use it as a CLI
    ```sh
    atlas_77 --help
    ```
3. Enjoy!
